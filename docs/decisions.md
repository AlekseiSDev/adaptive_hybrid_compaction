- **[2026-05-13] A4 hysteresis state — caller-owned, bidirectional, threshold=2**: `classifyWithHysteresis(features, prevState?): { class, newState }` — pure функция; `HysteresisState = { lastClass, pendingClass | null, pendingCount }`. Смена класса требует **2 последовательных turn'a** с тем же candidate (§3.2 verbatim), bidirectional (любая пара класса). Когда candidate ломает streak (intermediate turn с третьим классом) — counter сбрасывается и стартует новый pending. State хранит вызывающая сторона (A6 будет lay it рядом с `Tier2.classSignal`); core stateless, что упрощает testability и сохраняет cache-friendly семантику (state меняется только при флипе). - **[2026-05-13] A5 `compact()` — pure async function, caller-owned cross-turn state**: `compact(input: CompactInput): Promise<CompactResult>` принимает tier1/tier2/tier3 + scratchpad + опциональный `hysteresisState` + flags/thresholds/configuredClass + deps, возвращает assembled `Message[]` + новые tier'ы + `newHysteresisState` + accumulated `events: CoreEvent[]` + `cachePrefixBytes`. Никакого Compactor class / module-level state — caller (A6 adapter) держит HysteresisState (один на session), Scratchpad (один на session), AsyncBuffer (один на session). Same layered discipline что A4 hysteresis: cache-friendly + testable. Trade-off: caller должен явно прокидывать state; benefit: zero hidden mutation, parallel sessions trivial.

- **[2026-05-13] A5 cache invariance contract scoped to within-epoch (reflection = epoch boundary)**: §9 invariant "prefix bytewise stable" формально перерисован: `∀ turn i within same reflection epoch: prefix(compact(history_i)) bytewise == prefix(compact(history_{i-1}))`. Reflection — единственная operation которая ломает prefix. Когда `flags.REFLECTION && tokens(tier2.observations) >= REFLECTION_THRESHOLD`, `compact()` вызывает `reflect()`, заменяет `tier2.observations` сжатой версией, эмитит `CompactionEvent{type:'reflection'}`. Cache-killer, но observable. `cacheInvariance.test.ts` теперь имеет within-epoch (2 consecutive compact() → bytewise-equal prefix) + across-reflection (forced low threshold → exactly один reflection event) — 4 теста суммарно.
